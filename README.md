# Hospital-Patient-Analytics-PowerBI
Power BI dashboard analyzing hospital patient admissions, LOS, billing, age groups, and medical conditions.

**Overview**

This project is a Hospital Patient Analytics Dashboard built using Power BI to analyze patient admissions, length of stay (LOS), billing trends, age groups, and medical conditions.

**Key Insights**
1 Total admitted patients, doctors, rooms, and average billing.

2 Monthly admission trends by admission type.

3 Average length of stay (LOS) and patient age analysis.

4 Medical condition and age-group wise patient distribution.

5 Hospital-wise medical condition analysis.

**Tools Used**
Power BI
Microsoft Excel (Data Preparation)


**Purpose**
This dashboard was created as part of my healthcare analytics learning and portfolio, focusing on real-world hospital KPIs and effective data visualization.

 **How to Use**
1 Download the .pbix file

2 Open it in Power BI Desktop

3 Use slicers and filters to explore insights
